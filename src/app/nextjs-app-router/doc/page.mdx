# Nextjs App Router

[Reference](https://nextjs.org/docs)

## Overview

### Nested Layouts

LayoutComponentをページごとに指定することができ,ネストされていく

### Server Component

Componentはデフォルトで Server Component として扱われる

### Loading Component

`loading.tsx` を作成することで,ローディング時の Component を指定できる

※ `layout.ts` を async component にしても `loading.tsx` は反映されない
