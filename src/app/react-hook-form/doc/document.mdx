# React Hook Form

[Reference](https://react-hook-form.com/)

## Overview

ReactでのForm作成をいい感じにしてくれるライブラリ

- onChangeが全く発火しないので、パフォーマンスが良い
- UIライブラリを使用する場合はContollerを使用する
- FormProviderを使用することでネストされた入力Componentの値を取得できる
- FieldArrayを使用することで配列の入力を扱える

### 最も簡単なサンプル

```tsx
export const Basic: FC = () => {
  const { register, handleSubmit } = useForm();

  const onSubmit = async (data: FormValues) => {
    console.log(data);
  };

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input {...register('firstName')} />
      <input {...register('lastName')} />
      <input type="submit" />
    </form>
  );
};
```
